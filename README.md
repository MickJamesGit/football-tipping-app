# TippingPoint

A web application for NRL and AFL tipping.

3 modes of tipping:

- Normal: score 1 point for each winning tip you make.
- Margin: tip by margin to receive extra points. Lose points if you tip the margin and your team loses.
- Streak: pick 1 team each week to win. Biggest streak of the year wins. You can skip a week if you aren't confident in any teams that week.
